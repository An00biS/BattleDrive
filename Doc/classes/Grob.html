Class Grob

A grob is an object that encapsulates an organized set of sprites which together form a pseudo-3D rotating graphical entity, useful for games.

Attributes

noAngles number Number of visual angles this grob can present.
spriteIndex number The current visual angle (sprite index) ( from 1 to noAngles )
angle number Grob's angle in degrees, 0 being straight down.
pegs Table { { name, position/positions } } Array of pegs; A peg is a point defined on the grob with regard to it's rotation; There is either a single PegOffset for all angles or one PegOffset per angle; PegOffset is a {x,y,h} table, where x and y are the offset from pivot point and h is the height (y distance from ground level).
