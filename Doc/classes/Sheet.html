Class Sheet

Sheet is an universal container; It is movable and scalable, can create hierarchies; Can serve as a window, panel, button, scrollbar, whatever...

Attributes

relativePos Coordinates Position relative to parent sheet (in pixels)
absolutePos Coordinates Sheet's position relative to it's ancestor node; The coordinates are always relative to top left corner, no matter how the sheet is anchored.
anchor Table
anchor.horizontal edges.LEFT or edges.RIGHT
anchor.vertical edges.TOP or edges.BOTTOM
onMouseOver CallbackList
onMouseOut CallbackList
onMouseDown CallbackList
onMouseUp CallbackList
onDrag CallbackList
renderer CallbackList
mouseIsOver Boolean Flags if mouse is hovering this sheet.
mouseIsDown Boolean Flags if mouse is being pressed above this sheet.
parentChanged Function Callback which updates this sheet's position and size if parent element was somehow changed.
sheets List
name string Sheet name
