Class Desk

A Desk object is a root element of the sheet hierarchy; Also serves as a GUI manager object.

Attributes

buttons Table Buttons pressed above the currently pointed sheet
buttons.right Boolean tells if right mouse button is pressed
buttons.middle Boolean tells if middle mouse button is pressed
buttons.left Boolean tells if left mouse button is pressed
pointedSheet Sheet Sheet currently hovered by mouse.

Methods

mouseMoved ( newX, newY, oldX, oldY ) A callback for a mouse motion event.
mousePressed ( x, y, button ) Mouse button press or mouse wheel turn callback.
mouseReleased ( x, y, button ) Mouse button release callback.
draw () Draws the GUI
attachChild ( s ) Attaches a sheet into the tree.
newSheet ( _relativePosX, _relativePosY, _width, _height, _parentChanged, _horizontalAnchor, _verticalAnchor ) Creates a new Sheet object as a child of this desk.
removeSheet ( indexOrPointer ) Detaches a sheet object from this desk.
getPointedSheet () Fetches the Sheet which mouse currently points to
