Class SheetRenderer

Basic sheet renderer.

Attributes

contents table Table of SheetContent objects.
palette Palette Table with colors.
lineColorTL Color Outline color for top and left border.
lineColorBR Color Outline color for bottom and right border.
fillColor Color

Methods

mouseEvent ( mouseOver, mouseDown ) Informs the renderer about mouse activity.
sheetActivationEvent ( v ) Informs the renderer that the sheet has been (de)activated.
addContent ( c ) Adds a renderable content to this renderer; NOTE: MUST be only used on attached renderers, otherwise it fails.
