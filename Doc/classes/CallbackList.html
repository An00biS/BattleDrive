Class CallbackList

Keeps a list of executable values for a given sheet event.

Attributes

list Table The list of callable values.
sheet Sheet The Sheet object this callback list belongs to

Methods

add ( fn ) Adds a value "fn" in the list
remove ( fn ) Removes a value "fn" from the list
call () Executes all values in the order they were added; Each executed value recieves the sheet as argument
getList () Fetches a list of registered handlers
