Class SubGrob

attachChild ()
Attaches a child to sub grob.

Parameters

self
subGrob SubGrob The subgrob to attach
pegIndexOrName number/string The peg to attach the sub-grob to, counting from 1; Passing zero uses the grob's pivot as the peg (useful for shadows and ground-based effects).
drawPriority Grob's position in drawing order, counting from 1; 1 = highest priority (drawn first); 0 or nil = lowest priority (drawn last); Negative number means the grob won't be drawn (useful for shadows, which need to be drawn separately).

Return value:

Grob The previous grob on the specified peg.
