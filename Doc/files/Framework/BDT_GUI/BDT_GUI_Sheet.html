Tables

class Sheet
Sheet is an universal container; It is movable and scalable, can create hierarchies; Can serve as a window, panel, button, scrollbar, whatever... Fields
  • relativePos: Table {x,y} - Position relative to parent sheet (in pixels)
  • absolutePos: Table {x,y} - Sheet's position relative to it's ancestor node; The coordinates are always relative to top left corner, no matter how the sheet is anchored.
  • anchor: Table { horizontal, vertical } - This sheet's anchors. Defaults: TOP, LEFT
  • anchor.horizontal: edges.LEFT or edges.RIGHT
  • anchor.vertical: edges.TOP or edges.BOTTOM
  • onMouseOver: Table(CallbackList)
  • onMouseOut: Table(CallbackList)
  • onMouseDown: Table(CallbackList)
  • onMouseUp: Table(CallbackList)
  • onDrag: Table(CallbackList)
  • renderer: Table(SheetRenderer)
  • mouseIsOver: Boolean - Flags if mouse is hovering this sheet.
  • mouseIsDown: Boolean - Flags if mouse is being pressed above this sheet.
  • parentChanged: Function - Callback which updates this sheet's position and size if parent element was somehow changed.
  • name: string Sheet name
